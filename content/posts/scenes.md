## Scenes

**AirHockey**
* **AirHockey** — The main networked game scene for the [Air Hockey](/posts/airhockey/) game. Contains the NetworkManager, lobby UI, dedicated server lifecycle components, and all gameplay objects (ball, paddles, field, goals).
* **TutorialScenes** — A folder containing all the tutorial scenes. Some of these scenes must be run before the main game scene for calibration and setup. See [Tutorial Scenes](/posts/tutorialscenes/) for details.

**ShapeCatcher**
* **ShapeCatcher** — The main game scene for the [Shape Catcher](/posts/shapecatcher/) game. Currently runs in legacy single-machine mode (not yet networked).

Introduction #

AIRPLAY-frontend is the web frontend for the iGYM system.

Note: The frontend has not been used with the networked version of the game and is primarily associated with tutorial/non-networked scenes. It has not been maintained since the Unity 6 update and should be considered unsupported with the current architecture.

The frontend provides a web-based control interface accessible from a mobile device (via the VITE dev server) for managing the game system during on-site sessions. (Originally used at Packard’s Field, which no longer exists.)
